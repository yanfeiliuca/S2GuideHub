# S2GuideHub Official Promotional Image Gallery

This folder stores official/publicly accessible promotional images for Subnautica 2 that can be referenced by S2GuideHub when an official screenshot or press image is needed.

## Source

- IGDB Subnautica 2 Press kit: https://www.igdb.com/games/subnautica-2/presskit
- Steam Subnautica 2 store page for cross-checking official publisher/developer/copyright context: https://store.steampowered.com/app/1962700/_2

## Usage Rules

- These are official promotional or press-kit images, not CC0/public-domain assets.
- Copyright, trademarks, character/world designs, and game imagery remain with Unknown Worlds Entertainment and related rightsholders.
- Use them with clear context and attribution, for example: "Official promotional screenshot via IGDB press kit / Unknown Worlds Entertainment."
- Do not present these images as S2GuideHub-owned artwork.
- For original decorative visuals, keep using AI-generated concept images with the AI disclosure already used on the site.

## Files

Metadata lives in `manifest.json`. Each entry includes the local filename, original source image URL, source page, credit, and rights note.
