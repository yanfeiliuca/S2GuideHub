# S2GuideHub Official Promotional Image Gallery

This folder stores official/publicly accessible promotional images for Subnautica 2 that can be referenced by S2GuideHub when an official screenshot, store image, trailer poster, library artwork, localized capsule, or logo asset is needed.

## Source

- Steam Subnautica 2 official store page: https://store.steampowered.com/app/1962700/Subnautica_2/
- SteamDB asset index used only to locate Steam-hosted asset paths: https://steamdb.info/app/1962700/screenshots/

## Usage Rules

- These are official promotional or store-page images, not CC0/public-domain assets.
- Copyright, trademarks, character/world designs, and game imagery remain with Unknown Worlds Entertainment and related rightsholders.
- Use them with clear context and attribution, for example: "Official Subnautica 2 promotional image via Steam / Unknown Worlds Entertainment."
- Do not present these images as S2GuideHub-owned artwork.
- The AI-generated submarine route-planning image remains available only where explicitly kept by the site owner.

## Files

Metadata lives in `manifest.json`. Each entry includes the local filename, original source image URL, source page, credit, and rights note.
