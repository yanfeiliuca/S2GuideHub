Quick call

Prioritize the Tadpole once basic survival loops are stable

The Tadpole is the first Blueprint Atlas page because it changes how a player thinks about distance, oxygen safety, hostile water, and return planning. Official store copy names the Tadpole as the submersible used to traverse biomes, while the Early Access roadmap makes clear that vehicles and related systems may continue changing.

Treat this page as a complete route framework, not a final coordinate walkthrough. The public route should become coordinate-level only after S2GuideHub checks the fragment path, build chain, and material counts in the current game build.

Route plan

Recommended route structure before coordinates

Use this order for the first verified route pass. Each stop needs a source note and a last-checked date before it becomes a finished walkthrough.

01

Prepare

Bring the Scanner, a strong oxygen margin, a beacon habit, spare storage, and a return rule. The Tadpole route is not a first-dive route.

02

Scan fragments

Player-guide sources currently report that three Tadpole fragment scans unlock the blueprint. S2GuideHub still needs to verify the safest stop order.

03

Build the base chain

Source-reported setup includes a Moonpool, Tadpole Dock clearance, and Vehicle Fabricator. Confirm space and materials before committing.

04

Craft and test

Craft the Tadpole only after the material list is checked against the live build. Run a short shakedown trip before pushing deeper.

05

Log changes

Record build version, fragment route, component counts, and any patch-sensitive notes for future updates.

Common mistakes

  • Chasing distant fragments before oxygen safety is comfortable.
  • Building the base setup without enough vertical clearance for vehicle docking.
  • Crafting the vehicle without planning repair and emergency power.
  • Treating Early Access coordinates as permanent after patches.
